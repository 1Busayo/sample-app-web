Getting Started

For this tutorial, we will be referencing to a sample app located in this repository. To build with Zoom Developer tools, you need developer access for your Zoom account provided by your admin.
You also need your JWT credentials for building your Web Client SDK app. Learn how to generate one from this doc.

Note: For quick and easy prototyping of this demo app, we have included an undocumented function named generateSignature in index.js file. However, we highly encourage you to generate this signature on the server side and keep your API Key/Secret confidential and secure.

  1. Install Node.js/NPM and Git if you do not already have them installed on your computer.

  2. Download/Clone the sample app from GitHub by typing the following in your terminal:

     git clone git@github.com:zoom/sample-app-web.git

  3. Navigate to the directory where you downloaded the sample app.

  1. Config YOUR_API_KEY and YOUR_API_SECRET and replace it with your own REST API Key and Secret. Save the file. Navigate back to the root of the project

    Edit path/sample-app-web/CDN/js/index.js or path/sample-app-web/Local/js/index.js.

     var API_KEY = 'YOUR_API_KEY';
 var API_SECRET = 'YOUR_API_SECRET';

    Save the file and in your terminal, type:

     cd path/sample-app-web/CDN
 npm install
 npm run start

  2. Once that is complete, this will install a node module called http-server. After running npm run start command, visit http://localhost:9999 in your browser. You should now be seeing the sample app UI.

  3. Enter a Name and the Meeting Number of the meeting you scheduled. Click Join.
